# Deliverables Workspace

该目录用于统一管理 CIS 4190/5190 Final Project 的最终提交产物与过程日志。

## 目录约定

- `dataset/`：最终提交数据集与数据说明
- `model/`：最终最佳模型与模型元信息
- `report/`：最终报告正文与导出版本
- `figures/`：报告图表（指标曲线、数据统计图）
- `logs/`：训练与评测日志
- `manifests/`：提交清单、版本映射、校验记录
- `notes/`：临时笔记与待办
- `temp/`：中间文件（可清理）

## 使用原则

1. 所有“可提交内容”优先写入本目录。
2. 每次关键步骤（训练、评测、绘图）都在 `logs/` 记录结果。
3. 每个最终文件都在 `manifests/` 留一份清单映射。
4. `temp/` 仅放可再生文件，最终提交前清理。
